Add these to your pom.xml:

```
<repositories>
        <repository>
            <id>oss-snapshots</id>
            <name>Sonatype OSS Releases</name>
            <url>https://oss.sonatype.org/content/repositories/releases/</url>
        </repository>
    </repositories>
```
